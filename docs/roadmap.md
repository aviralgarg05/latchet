# Roadmap

## Phase 0-1

- Stable event schema and projection engine
- Local file store
- CLI for create/log/derive/show/verify/export/import/redact/diff
- MCP server
- Adapter contracts and prompt exports
- Fixture corpus and core tests

## Phase 2

- Codex, Claude Code, Gemini CLI, and Cursor integration packages
- Hook-based auto-append flows
- Session/log importers
- OpenTelemetry evidence ingestion
- Detailed execution plan in [Phase 2 Execution](phase-2-execution.md)

## Phase 3

- Destination-specific handoff packs
- Contradiction detection
- Cross-tool import/export hardening
- Better diff views across sessions

## Phase 4

- Optional hosted sync
- Shared workspaces and review workflows
- Governance and signed exports
- Search and analytics

## Phase 5

- Browser/chat adapters
- ChatGPT App and MCP-assisted flows where supported
- Best-effort web extraction
